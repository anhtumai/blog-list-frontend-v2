> Written in React (Typescript)

Link to course: [Course Url](https://fullstackopen.com/en/part7)

## Installation

```bash
$> git clone git@github.com:anhtumai/blog-list-frontend-v2.git
$> cd blog-list-frontend-v2
$> npm install
```

### Run

```bash
$> npm start
```

### Test

#### 1. Cypress

```bash
$> npm run cypress:open
```
